MVVM Life Cycle
================

Unify activity and fragment life cycle for MVVM architecture


## Installation

```
compile "me.henrytao:mvvm-life-cycle:<latest-version>"
```

`mvvm-life-cycle` is deployed to `jCenter`. Make sure you have `jcenter()` in your project gradle.
